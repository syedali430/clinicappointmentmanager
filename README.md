# Clinic Appointment Manager

Clinic Appointment Manager is a Java 8 desktop application developed using Swing and Maven following the MVC architectural pattern.

## Functionalities

- Add appointments
- Update appointments
- Delete appointments
- View appointments

Each appointment contains:
- Appointment ID
- Patient Name
- Doctor Name
- Appointment Date
- Appointment Time
- Reason

## Application Features

- Appointment management with complete CRUD operations
- MongoDB integration for data storage
- Test-Driven Development (TDD) approach
- Full code coverage analysis using SonarCloud and Coveralls
- Continuous Integration using GitHub Actions across multiple Java environments

## Technologies Used

- Java 8
- Maven
- Swing
- MongoDB
- JUnit 4
- Mockito
- AssertJ
- AssertJ Swing
- Awaitility
